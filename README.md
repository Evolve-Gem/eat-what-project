# 吃什么呢：基于微信小程序的校园饮食推荐系统

一个面向大学生校园饮食决策场景的小程序项目，支持基于心情、口味、时间的规则推荐，并拓展了基于 DeepSeek API 的 AI 自然语言推荐能力。

---

## 项目简介

“吃什么呢”是一款面向大学生校园生活场景的饮食推荐小程序，针对学生日常“今天吃什么”选择困难、食堂窗口选择成本高、饮食偏好匹配不明确等问题，提供基于心情、口味、时间等维度的餐食推荐功能。

项目最初从 Python 规则推荐逻辑开始，逐步完成了微信小程序前端、Flask 后端接口、云端部署、小程序上线和 AI 能力接入，形成了一个从需求分析到产品落地的完整应用闭环。

后续项目接入 DeepSeek API，新增“AI帮我选”入口，用户可以用一句自然语言描述自己的饮食需求，系统会生成推荐菜品、推荐理由和自然语言解释。

---

## 项目亮点

- 完成微信小程序前端、Flask 后端、云端部署与小程序版本上传，形成完整项目闭环
- 支持基于心情、口味、出餐时间的规则推荐，并返回匹配分数与推荐理由
- 接入 DeepSeek API，实现自然语言饮食需求理解与 AI 推荐结果生成
- 新增 AI 推荐结果页，展示用户原始输入、AI 理解到的需求、推荐菜品、推荐理由和解释文本
- 针对小程序上传包体积限制，对图片资源进行压缩，并实现菜品图片本地化匹配
- 项目已完成小程序基础版本上线，AI 增强版作为 Demo 版本用于项目展示与后续扩展

## 功能模块

### 1. 基础规则推荐

用户可以根据当前状态进行结构化选择：

- 心情：开心、解压、自律、普通、赶时间等
- 口味：辣、清淡、油炸、健康、家常等
- 时间：快、中、慢

系统根据用户选择与菜品标签进行匹配，返回推荐菜品、价格、匹配分数和推荐理由。

---

### 2. AI 自然语言推荐

用户可以直接输入一句自然语言，例如：

> 我今天有点累，想吃点热乎的，别太贵。

系统会调用 DeepSeek API，对用户需求进行理解，并返回：

- AI 理解到的需求
- 推荐菜品
- 推荐分数
- 推荐理由
- AI 自然语言解释
- 对应菜品图片

---

### 3. 菜品图片匹配

为了提升小程序展示效果，项目对菜品图片进行了压缩和本地化管理，并根据 AI 返回的菜品名称自动匹配对应图片。

示例：

| 菜品 | 图片 |
|---|---|
| 麻辣香锅 | `malaxiangguo.png` |
| 西红柿鸡蛋面 | `tomato-egg-noodle.png` |
| 鸡胸肉沙拉 | `chicken-salad.png` |
| 炸鸡汉堡 | `burger.png` |
| 黄焖鸡 | `huang-men-ji.png` |
| 轻食三明治 | `sandwich.png` |
